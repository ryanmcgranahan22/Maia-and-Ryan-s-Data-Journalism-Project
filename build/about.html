Useful Definitions

Death by unnatural causes:
An untimely death brought about by methods other than natural, bodily causes, such as sickness or old age

Cause of death by unnatural causes:
The tool or action that brought about an unnatural death such as a cut, a firearm, or a fire.

Ratio:
The amount of people who die by unnatural causes in a given state divided by the amount of people who live in the state

Macrodata:
The data describing a big territory, here, the united states

Microdata:
The data describing a smaller territory, here, a specific state

Final Design Elements

Based off of the evaluations above, we have decided to include the folliwng elements in our final design:

Macro Design

  1. The map from Maia's sketch because using shades instead of colors allows for accessability/perceptibility for colorblind people.
  2. It also means that the map will show amount of violent deaths rather than mechanism (as we will discuss mechnism for the micro page)
  3. The key from Maia's sketch as it makes the page more understandable.
  4. The paragraph from Ryan's sketch as it helps with understandability
  5. The buttons from Maia's sketch as they are great sigmifiers
  6. A highlight around each state that you hover over to signify that you can click on themselves

Micro Design

  1. The pie chart in Maia's sketch instead of the table in Ryan's sketch because it is more perceptible and more visual-based than word-based
  2. The colored-in state from Ryan's sketch because it provides context for the pie chart.
  3. A key for each of the diagrams.
  4. The buttons from Maia's sketch because they work as great signifiers.
  5. The paragraphs/titles from Maia's sketch because they make the page more understandable.


Project Pipeline

Division of Labor

  • Maia will focus on page generation, data analysis, and bootstrap.
  • Ryan will focus on map/chart generation and css

Pipeline Steps

  1. create fs.js to turn data into a json.
  2. use ejs.js to create a macropage with all the data
  3. use a for loop in ejs.js to create micropages using ejs.js
  4. create tables on both the macro/micro pages showing the relavent data for that page
  5. add a header using bootstrap to link all the pages to each other
  6. add "previous" and "next" buttons to the micropages to link them together
  7. add the diagrams to each page using plotly and jquery
  8. add dynamic descriptions to each page that use variables from the data
  9. use css to make it pretty
